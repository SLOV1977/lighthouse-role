Lighthouse
=========

Ansible-роль, которая устанавливает и настраивает LightHouse (веб-интерфейс для ClickHouse) через Nginx.

Переменные используемые ролью
-----------------------------

| Переменные | Описание |
|---------------|----------------|
| lighthouse_root_dir | Размещение клиента Lighthouse |

Пример использования роли
-------------------------

    - hosts: lighthouse
      roles:
        - lighthouse-role
